# Data Cleaning & Visualization – Titanic Dataset

Project as part of internship: data preprocessing and exploratory data analysis on the classic Titanic survival dataset.

## Overview
- Cleaned the dataset: handled missing values, removed duplicates, fixed data types.
- Engineered a `FamilySize` feature.
- Created 5 visualizations to uncover survival patterns.

## Key Insights
- Women had a significantly higher survival rate.
- First-class passengers survived more than second and third class.
- Fare was heavily right-skewed; most passengers paid low fares.
- Age varied by class, with first-class passengers being older.

## Tools Used
- Python (Pandas, NumPy)
- Visualization: Matplotlib, Seaborn
- Environment: Google Colab

## Data Source
[Titanic Dataset](https://www.kaggle.com/datasets/yasserh/titanic-dataset) from Kaggle.

## How to Run
1. Clone the repository or download the files.
2. Open the `.ipynb` file in Google Colab or Jupyter Notebook.
3. Ensure the dataset `Titanic-Dataset.csv` is in the same directory (or adjust the path).
4. Run all cells.
